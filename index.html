What Agar Serverbrowser is

The Agar Serverbrowser is a Userscript written and maintained by @Freigeist, for simple browsing and joining Agar.io private servers.

Installation

Firefox users will need to install the Greasemonkey. Google Chrome or Opera users might want to download the Tampermonkey respectively Violent monkey extensions. Internet Explorer / Edge users... ehm... please get a real browser.

After installing the extensions, you should be able to install the script by clicking this link
